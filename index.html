About Me & Philosophy

I am a High-Performance Software Engineer specializing in the intersection of Artificial Intelligence, Signal Processing, and Hardware Acceleration. My work is centered on the "last mile" of performance—transforming complex, high-level algorithms into deterministic, production-grade systems that run on specialized silicon.

I believe the next generation of AI isn't just about bigger models, but about spatial and modal awareness. Whether it's guiding an MVDR Beamformer with visual coordinates or eliminating 2-second latency gaps in LLM speech via custom C++ kernels, my goal is to break the silos between the application layer and the hardware. I build systems where every clock cycle is accounted for, and every microsecond of latency is earned.

Case Studies: Performance Solved

SPEECH RECOGNITION (ASR)

Eliminating Homophone Errors in ASR

Implemented a Phonetic AEC (Automatic Error Correction) layer using pinyin-based fuzzy matching. This ensures industrial-grade precision for technical terms by correcting phonetic misidentifications in real-time streaming.

AUDIO FUSION (VisuBeam)

Cross-modal Steering: Visual-Aided Beamforming

Developed a fusion engine that uses Visual Tracking Coordinates to guide an MVDR Beamformer. By decoupling I/O via a Shared Circular Buffer, the system achieves 45°→70° smooth spatial tracking with zero-drop audio capture.

SYSTEM ARCHITECTURE (VisuBeam)

Zero-Latency Multi-modal Pipeline

Engineered a production-grade DSP stack that orchestrates WPE, APM, and MVDR algorithms in a non-blocking fan-out architecture. By implementing Lock-free Shared Buffering and a custom Binary Hybrid Protocol, the system achieves sub-10ms processing overhead for deterministic real-time AI interaction.

LLM OPTIMIZATION (TTS)

Solving the 2s Latency Gap in AI Speech

Through specialized forward-pass monkey-patching and raw PCM binary streaming, I achieved a TTFB of ~380ms, making AI interaction truly conversational.
