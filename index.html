Engineering Philosophy

I am a software engineer driven by the pursuit of computational efficiency. In an era of bloated software, I focus on the "last mile" of performance—where high-level algorithms meet low-level silicon.

My work spans from optimizing Large Language Models (LLMs) for real-time speech interaction to writing C++ JNI wrappers that squeeze every drop of performance out of dedicated hardware like Rockchip's RGA3 cores. I believe that true innovation happens when you understand the entire stack, from the Python application layer down to the kernel memory management.

Eliminating Homophone Errors in ASR

Implemented a Phonetic AEC (Automatic Error Correction) layer using pinyin-based fuzzy matching. This ensures industrial-grade precision for technical terms by correcting phonetic misidentifications in real-time streaming.

AUDIO FUSION (VisuBeam)

Cross-modal Steering: Visual-Aided Beamforming

Developed a fusion engine that uses Visual Tracking Coordinates to guide an MVDR Beamformer. By decoupling I/O via a Shared Circular Buffer, the system achieves 45°→70° smooth spatial tracking with zero-drop audio capture.

CONCURRENCY (SPEC-009)

Multi-Consumer Fan-out Architecture

Engineered a thread-safe Shared Buffer system that allows a single microphone stream to be consumed by multiple independent enhancement instances and a DOA engine simultaneously, with DTLN Model Pooling for state isolation.

Solving the 2s Latency Gap in AI Speech

Through specialized forward-pass monkey-patching and raw PCM binary streaming, I achieved a TTFB of ~380ms, making AI interaction truly conversational.
